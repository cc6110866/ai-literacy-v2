# AI识字伴侣 - 开发任务清单

> 最后更新：2026-04-02
> 原则：以 D1 实际表结构（D1-SCHEMA.md）为准进行开发

---

## 🔴 P0 — 必须补的核心功能

### [ ] 发音/TTS 朗读
- **描述**：儿童识字必须能听发音，拼音+汉字朗读
- **数据**：Character 表已有 `audioUrl` 列（部分有值），`pinyin` 和 `tone` 列全量
- **方案**：
  - 优先用 `audioUrl` 播放预录音频
  - 无音频的字用 Web Speech API (SpeechSynthesis) 做 TTS
  - 识字页、练习页、复习页都需要发音
- **工作量**：中等（1-2天）

### [ ] 笔顺展示
- **描述**：展示汉字书写笔画顺序，竞品标配
- **数据**：Character 表已有 `strokes`（笔画数）、`strokeGif`（笔画GIF URL）列
- **方案**：
  - 有 `strokeGif` 的字直接播放 GIF 动画
  - 无 GIF 的字用 Canvas + 笔顺数据绘制（需接入笔顺数据源如 HanziWriter）
  - 识字页字卡区域增加"看笔顺"按钮
- **依赖**：评估 HanziWriter（开源库，含 9000+ 字笔顺数据）是否可集成
- **工作量**：中等（1-2天）

### [ ] 音效反馈
- **描述**：答对/答错/操作时有声音反馈
- **方案**：
  - 用 Web Audio API 生成简单音效（不需要音频文件）
  - 答对：清脆"叮"声
  - 答错：低沉"嘟"声
  - 点按钮：轻微"咔"声
  - 家长中心可开关音效
- **工作量**：小（半天）

## 🟡 P1 — 重要体验优化

### [ ] 新用户引导
- **描述**：首次进入时的 3 步引导动画
- **方案**：
  - Step 1："欢迎来到 AI 识字伴侣！" + mascot 动画
  - Step 2："点字卡就能学新字，试试看！" + 手势指引
  - Step 3："每天 5 分钟，轻松识字！" + 开始按钮
  - localStorage `ai-literacy-onboarded` 标记已完成
- **工作量**：小（半天）

### [ ] 练习形式多样化
- **描述**：当前只有选拼音/选汉字，需要增加玩法
- **方案**（逐步增加）：
  - **听音选字**：播放发音，选出对应汉字（需先完成 TTS）
  - **汉字消消乐**：相同拼音的字配对消除
  - **组词配对**：字+字→词，拖拽配对
- **工作量**：每种 1-2 天

### [ ] 学习提醒
- **描述**：家长设置学习时间，到点提醒
- **方案**：
  - 家长中心增加"学习提醒"设置（时间 + 频率）
  - 使用 Web Notification API（需用户授权）
  - 到点弹出通知："小老虎提醒你该识字啦！"
  - 页面打开时检查并请求通知权限
- **工作量**：小（半天）

### [ ] 答错针对性练习
- **描述**：分析错题数据，自动出针对性练习
- **方案**：
  - 练习页结果页"需要复习的字"可直接生成专项练习
  - 新增"错题本"页面，列出所有答错过的字
  - 错题优先出现在复习队列
- **工作量**：中等（1天）

## 🟢 P2 — 锦上添花

### [ ] 数据同步/登录
- **描述**：换设备/清浏览器不丢数据
- **方案**：
  - 微信一键登录（需要后端 + 微信开放平台）
  - 或手机号验证码登录
  - 登录后 localStorage 数据同步到 D1
  - 未登录时保持纯本地模式
- **工作量**：大（3-5天）

### [ ] 拍照识字
- **描述**：拍照识别汉字，查看释义
- **方案**：
  - 调用 OCR API 识别图片中的汉字
  - 匹配字库，展示释义/拼音/笔顺
  - 支持相册选图和实时拍照
- **依赖**：需要 OCR API（百度/腾讯/Google Vision）
- **工作量**：中等（1-2天）

### [ ] 分级阅读
- **描述**：用已学字生成简单句子/短文让孩子读
- **方案**：
  - 根据孩子已学字，从预设语料库筛选"全认识"的句子
  - 逐句展示，点击生字可看释义
  - 读完统计阅读字数
- **依赖**：需要准备阅读语料库
- **工作量**：大（2-3天）

### [ ] AI 个性化学习路径
- **描述**：根据学习数据智能调整学习内容和节奏
- **方案**：
  - 分析正确率低的 category/level，加强出题
  - 学习速度快的自动增加每日目标
  - 连续答对多的字减少复习频率
- **工作量**：大（3-5天）

### [ ] 暗色模式
- **描述**：支持深色主题，保护眼睛
- **方案**：Tailwind dark mode，跟随系统设置
- **工作量**：小（1天）

### [ ] 离线支持
- **描述**：无网络时仍可学习
- **方案**：Service Worker 缓存字库数据和静态资源
- **工作量**：中等（1-2天）

### [ ] 多用户支持
- **描述**：一个设备多个孩子使用
- **方案**：添加用户切换功能，每个用户独立数据
- **工作量**：中等（1天）

### [ ] 汉字搜索
- **描述**：搜索字库查看释义/笔顺
- **方案**：首页或家长中心添加搜索入口，支持拼音/汉字/释义搜索
- **工作量**：小（半天）
